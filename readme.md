## Go Bookmarks

A simple example GUI app written in Go using the [Fyne](https://fyne.io) toolkit.

Install & Run
-------------

Install
```
git clone https://github.com/yourusername/go-bookmarks.git
cd go-bookmarks
go mod tidy
```

Run the app:
```
go run .
```

Build a binary (optional):
```
go build -o go-bookmarks
```
